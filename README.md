# Tic-Tac-Toe

## Run Locally

Clone the project

```bash
  git clone hhttps://github.com/Cristian-Baeza/TTT-Ruby
```

Go to the project directory

```bash
  cd TTT-Ruby
```

Install dependencies

```bash
  bundle install
```

Start application

```bash
  ruby lib/runner.rb
```

## Running Tests

From root directory

```bash
  cd TTT-Ruby
```

Run rspec on a file from spec directory

```bash
  rspec spec/*test-file*.rb
```
