# tic-tac-toe

## Run Locally

Clone the project

```bash
  git clone hhttps://github.com/Cristian-Baeza/TTT-Ruby
```

Go to the project directory

```bash
  cd TTT-Ruby
```

Install dependencies

```bash
  gem install
```

Start application

```bash
  cd lib
  ruby tictactoe.rb
```

## Running Tests

Go to root directory

```bash
  cd TTT-Ruby
```

Run rspec on spec file

```bash
  rspec spec/tictactoe.rb
```
